Product definition

    name (string)
    description (string)
    price (float)
    image (file)
    belongs to n categories

Category definition

    name (string)
    parent category (null|category)

Features

    Ability to create a product (from cli)
    A listing products with ability to sort by price, or/and filter by a category (from web)

Testing
